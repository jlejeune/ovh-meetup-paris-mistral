Why Mistral ?

  • Uniformity with public service (configuration mastering)
  • Easy access to entire OpenStack API through the workflows
  • Can have workers in different zones (vs StackStorm)
  • Powerful workflow engine
  • YAML definition easy to version
  • Custom actions can be written easily in Python
